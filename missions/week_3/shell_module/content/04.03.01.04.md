# 🐚 Introduction to Shell Basics (Wildcards)

# 🧵 Character Ranges `[ ]`

Square brackets match **one character** from a specified range or set.

> 📌 Examples

```bash
ls n[1-3]*.cfg
```

* Matches files starting with `n1`, `n2`, or `n3`.

```bash
ls [a-z]*.txt
```

* Matches files that begin with any lowercase letter.
